README: Keyboard Controls

Select Object:
Camera = c
Merry Go Round = m
Light Post = l
Moon = 0
Rain = Not Implemented (Each Raindrop has its own CF, that's a lot of CF's)

Rotate Object:
Shift + Arrow Keys (Use combination of Up/Down and Left/Right to move along Z-Axis)

Translate Object:
X = X-Axis
Y = Y-Axis
Z = Z-Axis

Add Wind:
S, D, Shift + S, Shift + D

Light Switches:
Light Post = 1
Moon = 0
